The CIO of Client.Inc  must decide whether to implement a new ERP system. An ERP system consisting of 5 modules is being considered.  The implementation cost for each module is 10M dollars.  However, there is uncertainty about whether the employees would accept and use the new system.  If the new system is well adopted, it is expected to generate efficiency gains of about $200M dollars; however, if the new system faces significant resistance, it will only generate a minimal benefit of $10M dollars. Client.Inc estimates that the probability of successful adoption is 0.5.

The CIO of Client.Inc has two alternatives. First, she can try to implement the whole system (5 modules) simultaneously. Second, she can choose to just implement one module first, and see whether the employees adopt it well.  If it is not well adopted, Client.Inc can choose to not implement the other 4 modules.  The cost to implement one module is still $10M dollars, and implementing just one module produces no meaningful benefits/savings by itself.   It is assumed that the outcome of one module implementation perfectly reflects the employees’ adoption of the whole system.  Evaluate if there is value in the one-module implementation experiment?
